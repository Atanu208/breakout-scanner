# Stock Breakout Scanner

AI-powered Stock Breakout Scanner built using Antigravity with a vibe coding approach.

This project scans Nifty 500 stocks and identifies 52-week high breakout opportunities through a clean, modern, and easy-to-use interface.


## 🚀 Features

- Detects 52-week high breakout stocks
- Covers Nifty 500 companies
- Clean and structured UI
- Fast stock scanning experience
- Built using AI-assisted vibe coding workflow
- Beginner-friendly and easy to navigate

---

## 📦 Installation

### 1. Install dependencies

```bash
npm install
````

---

## ▶️ Run Project

```bash
npm start
```

Open in browser:

```bash
http://localhost:2000
```

---

## 📌 Usage

1. Start the application
2. Scan Nifty 500 stocks
3. View stocks hitting 52-week highs
4. Analyze breakout opportunities quickly

## 💡 Purpose

This project helps traders and investors quickly identify momentum stocks making fresh yearly highs using automation and a modern dashboard experience.
